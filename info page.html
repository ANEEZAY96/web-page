web development

web development is the work involved in developing a website for the internet. web development can range from developing a simple single static page of plain text to complex web applications, electronic business, and social netwerk services.web development is a way to make people aware of the services or products you ara offering.

types of web development

1-front end web

front end web development is the development of graphical user interference of a website, through the use of HTML,CSS,JAVASCRIPT,so that users can view and interact with that website.

2-back end web

it is also known as serverside development. it is everything that users don't see and contains behind the scenes activities that occur when performing any action on website.it focuses primarilly on databases,backend logic and servers.

3-full stack web development

it refers to the development of both front end(client side0 and back end(server side) portion of web application. full stack web developers have the ability to design complete web applications and websites.
